# Tutorials

scDLKit is most useful when the tutorials are easy to find and easy to run. This page groups the Jupyter notebooks and script-based demos into a researcher-first workflow.

## Tutorial map

| Workflow | Best for | Extra install | Entry point |
| --- | --- | --- | --- |
| Beginner notebook | First-time users, students, quick sanity checks | `scdlkit[notebook]` | `examples/first_run_synthetic.ipynb` |
| Beginner script | Fast terminal smoke run | none | `examples/first_run_synthetic.py` |
| PBMC VAE notebook | Representation learning on real single-cell data | `scdlkit[scanpy]` | `examples/train_vae_pbmc.ipynb` |
| PBMC comparison notebook | Benchmarking AE, VAE, and Transformer AE | `scdlkit[scanpy]` | `examples/compare_models_pbmc.ipynb` |
| Classification notebook | Supervised label prediction workflow | `scdlkit[scanpy]` | `examples/classification_demo.ipynb` |

## Beginner notebook

This is the first tutorial most users should open.

What it covers:

- creating a small synthetic `AnnData`
- fitting a baseline model with `TaskRunner`
- evaluating metrics
- saving a loss curve, latent projection, and Markdown report

Run it locally:

```bash
python -m pip install "scdlkit[notebook]"
jupyter notebook examples/first_run_synthetic.ipynb
```

Repository links:

- [Open on GitHub](https://github.com/uddamvathanak/scDLKit/blob/main/examples/first_run_synthetic.ipynb)
- [Download notebook](https://raw.githubusercontent.com/uddamvathanak/scDLKit/main/examples/first_run_synthetic.ipynb)

## Script alternative

If you want the same first-run story without opening Jupyter:

```bash
python examples/first_run_synthetic.py
```

This writes outputs to `artifacts/first_run/`.

## PBMC notebooks

These tutorials move from synthetic data to a more realistic single-cell workflow. Install the Scanpy extra first:

```bash
python -m pip install "scdlkit[scanpy]"
```

Available notebooks:

- [PBMC VAE walkthrough](https://github.com/uddamvathanak/scDLKit/blob/main/examples/train_vae_pbmc.ipynb)
- [PBMC model comparison](https://github.com/uddamvathanak/scDLKit/blob/main/examples/compare_models_pbmc.ipynb)
- [PBMC classification demo](https://github.com/uddamvathanak/scDLKit/blob/main/examples/classification_demo.ipynb)

## Suggested learning order

1. Start with `examples/first_run_synthetic.ipynb`
2. Move to `examples/train_vae_pbmc.ipynb`
3. Compare models with `examples/compare_models_pbmc.ipynb`
4. Use `examples/classification_demo.ipynb` for a supervised workflow
