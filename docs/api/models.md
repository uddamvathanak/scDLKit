# Built-in models

The built-in models are secondary to the main workflow APIs, but it is still useful to know what the registry exposes.

Current stable built-ins:

- `autoencoder`
- `vae`
- `denoising_autoencoder`
- `transformer_ae`
- `mlp_classifier`

Use this page to inspect constructor parameters and supported tasks. Use the tutorials and guides when you want help choosing between them.
