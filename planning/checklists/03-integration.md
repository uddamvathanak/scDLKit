# Milestone 3: integration pillar

Status: planned

## Objective

Define integration and representation-transfer benchmarking as a real task
pillar with task-specific metrics and datasets.

## Why this matters for the paper

Integration is one of the four main research tasks. It should not be reduced to
generic embedding visualization.

## Current state

- representation workflows exist
- integration benchmarking does not yet exist as a formal task pipeline

## Exit artifacts

- integration task spec
- metric pipeline
- dataset shortlist
- first integration tutorial
